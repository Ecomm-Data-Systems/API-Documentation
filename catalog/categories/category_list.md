# Catalog::Categories::Category List

Retrieve a list of product categories

GET:
```bash
/api/v1/product_categories
```

Response example
```json

{
    "categories": [
        {
            "name": "Computers & Games",
            "apikey": "j0gcEpRddG859StfDWD1yQ"
        },
        {
            "name": "Garden & Shoes",
            "apikey": "y42ZUktRiq_u36Dv1W8Xjg"
        },...
    ]
}

```
